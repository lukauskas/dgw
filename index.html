Dynamic Genome Warping (DGW)

This is the homepage for the open source DGW library developped by Saulius Lukauskas.

Stable binaries of DGW are distributed over PyPi and you should be able to install it using setuptools:

easy_install dgw

If this fails, consult the Installation section in the documentation for a quick guide on how to install it.

Specifically, have a read through the Quick Start Guide for a guide on how to get started with DGW, or consult the full documentation at https://dgw.readthedocs.org.
